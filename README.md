# 🍽️ Cookbook Application

Backend that allows users to browse and manage cooking recipes.

## 🚀 Tech Stack
- Frontend: React, TypeScript
- Backend: Java, Spring Boot
- Database: PostgreSQL
- API: REST

## ✨ Features
- Recipe list
- Recipe details page
- CRUD operations
- REST API integration

## ⚙️ Getting Started

docker-compose up
